Página Inicial

Este sistema demonstra o resultado de algumas formas de fazer chat que eu implementei faz alguns anos. Selecione qual chat deseja ver o resultado pelo menu do cabeçalho do sistema.

Para ver o chat funcionando: Abra duas abas no seu navegador com o mesmo tipo de chat, depois envie uma mensagem por meio de uma das abas, em seguida vá na outra aba e veja que a mensagem aparece sem ter que atualizar a página.

No primeiro link, Ajax Reverso, o sistema mostra o resultado do chat feito utilizando long polling que consiste em uma requisição ajax normal para uma função PHP que não responde de imediato.

A função PHP que criei para isso dei o nome de carregar_chat_ajax_reverso, esta possui um loop que só desarma quando detecta que há uma nova modificação em um arquivo txt (arquivo de texto).

No segundo link, Web Socket, o sistema mostra o resultado do chat feito com Web Sockets. Eu fiz um arquivo que chamei de manipulador_de_web_sockets.php e para utilizar é necessário ter a extensão sockets (ou php_sockets) ativada no php.ini, ir na pasta App\Libraries deste sistema e executar o comando:

php -q manipulador_de_web_sockets.php

Este arquivo vai manipular as conexões de websocket no lado do servidor (backend), decodificando o que é recebido pelo servidor e codificando o que é enviado pelo servidor. Além disso possui um loop que identifica qual dos sockets está com informação e envia a informação deste socket para todos os outros conectados.
